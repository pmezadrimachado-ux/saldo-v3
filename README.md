# Saldo V3

Aplicativo financeiro pessoal, mobile first, offline first e instalável como PWA.

## Status

Release Candidate 4 — validação final da V3.

Esta versão ainda não possui funcionalidades financeiras reais. Ela entrega a base técnica do projeto:

- estrutura de pastas;
- bootstrap;
- router;
- estado global;
- event bus;
- error boundary;
- app shell;
- páginas placeholder;
- PWA inicial;
- compatibilidade com GitHub Pages.

## Como executar localmente

```bash
py -m http.server 8000
```

Acesse:

```text
http://localhost:8000
```
